# Cricket World Cup Insights  
BI & Data Visualization Project using Google Looker Studio

# Project Overview
This project analyzes **historical Cricket World Cup data** to uncover meaningful insights related to **team performance, player analytics, trends, and venue effectiveness**.  
The goal is to demonstrate **Business Intelligence (BI)** and **data visualization skills** using **Google Looker Studio**.

An interactive dashboard is built to help users explore performance patterns across countries, players, and venues.

---

## Objectives
- Analyze country-wise performance in Cricket World Cups
- Compare player performances using interactive filters
- Visualize trends in highest individual scores over time
- Identify the best-performing venues for the Indian cricket team
- Build an interactive BI dashboard for decision support

---

## Dataset Description
The project uses structured CSV files derived from publicly available cricket statistics (Wikipedia / ESPNcricinfo-style data).

## Files Used

Country-wise match performance 
Player runs and highest scores 
Bowling extras (wides, no-balls) 
Venue-wise performance for India 

##  Tools & Technologies
- Google Looker Studio – Dashboard & visualization
- CSV Files – Structured data storage
- Wikipedia – Data reference sources
